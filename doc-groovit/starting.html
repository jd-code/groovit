Starting groovit.
  Here we don't deal with groovit compilation and installation.
  Groovit should work in any ANSI emulator terminal. It includes mainly linux console on one hand, and xterm-like stuff on the other hand.
  In linux console you only have to type-in "groovit" (supposing it IS in your PATH) and it should start.
  With X11, the best is to have the DISPLAY global environment var set to the appropriate value then to type-in "groovit". If the DISPLAY-var is correctly set, groovit forks into a new xterm, and outputs the error and info messages into the initial terminal.
